# Conflict Mediation

## Purpose

Help the user understand and navigate a conflict by identifying the underlying value-frame mismatch. Most stuck conflicts are not about facts — they are about incompatible moral frameworks that neither party has made explicit.

## When to use

The user says something like:
- "We keep arguing about the same thing"
- "They just don't get it"
- "How do I resolve this disagreement?"
- "We can't agree on what's fair"
- "This team/relationship is stuck"
- "Why do they see this so differently?"

## Workflow

### 1. Map the positions

State each party's position in one sentence. If the user only presents their side, ask: "What would the other person say their reason is?" Do not accept "they're just wrong" — dig for the actual frame.

### 2. Identify the value-frame mismatch

Most conflicts map to one or more of these frame collisions:

| Frame A | vs | Frame B |
|---------|-----|---------|
| Fairness | vs | Loyalty |
| Autonomy | vs | Safety |
| Truth | vs | Harmony |
| Dignity | vs | Efficiency |
| Individual rights | vs | Collective good |
| Principle | vs | Pragmatism |
| Innovation | vs | Stability |
| Merit | vs | Care |

Name the collision explicitly: "You are operating from a fairness frame. They are operating from a loyalty frame. Neither is wrong — they are answering different questions."

### 3. Apply lenses (select 2-3)

**Default set:** Aristotelian + Kantian + Existentialist

Add or swap based on the conflict:
- If one party has structural power over the other: add Foucauldian
- If both parties are reacting from hurt or ego: add Buddhist
- If one party is performing moral superiority: add Nietzschean
- If the conflict is about practical outcomes: swap in Utilitarian

For each lens:
- What does this lens say is the real source of the conflict?
- What would resolution look like through this lens?
- What would this lens sacrifice that the other party won't accept?

### 4. Translation

The highest-value move in mediation is translation, not judgment. Produce a statement that:
- Validates what each side actually cares about (not their position, their underlying value)
- Names the tension honestly without pretending it doesn't exist
- Shows each side what they would need to give up for resolution

Format:

> "Party A cares about [value]. Party B cares about [value]. The conflict is stuck because [specific mechanism]. Resolution requires [what each side would need to accept]."

### 5. Path forward

Offer 2-3 concrete options, ranked:

1. **Best resolution:** What would it look like if both frames were partially honored?
2. **Workable compromise:** What is the least costly concession for each side?
3. **Clean separation:** If the values are genuinely incompatible, what does a respectful divergence look like?

### 6. What might keep this stuck

Name the most likely reason this conflict won't resolve even with good analysis. Usually: one party won't name their real concern, the power dynamic prevents honest negotiation, or the conflict serves a function neither party wants to admit.

## Output shape

- 400-800 words total
- Name the frame mismatch early
- Translate, don't judge
- End with concrete options, not platitudes
