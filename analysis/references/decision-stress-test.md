# Decision Stress Test

## Purpose

Help the user pressure-test a decision from multiple angles before committing. The goal is not to paralyze but to surface what they haven't considered and strengthen their conviction or change their mind.

## When to use

The user says something like:
- "Should I...?"
- "I'm trying to decide between..."
- "Help me think through this"
- "Stress-test this plan"
- "What am I missing?"

## Workflow

### 1. Extract the decision

State the decision back to the user in one sentence. If it's unclear, ask one question to clarify. Do not proceed until the core choice is named.

### 2. Identify what's actually at stake

Before selecting lenses, identify:
- **Reversibility:** Can this be undone? How easily? At what cost?
- **Time horizon:** Is this a one-time choice or does it compound?
- **Who is affected:** Just the user, or others too?
- **What the user fears:** The real anxiety, not the stated one.

### 3. Apply lenses (select 3-4)

**Default set:** Stoic + Utilitarian + Nietzschean

Add or swap based on what's at stake:
- If character or identity is central: add Aristotelian
- If the user is stuck in anxiety or worst-case spiraling: lead with Stoic
- If the user's stated reason feels thin: lead with Nietzschean
- If the decision affects many people: lead with Utilitarian
- If the user says "I have no choice": lead with Existentialist
- If the user is clinging to an outcome: add Buddhist

For each lens, answer:
- What does this lens say you should do?
- What does it see that the other lenses miss?
- Where might it lead you astray?

### 4. Convergence and tension

- Where do the lenses agree? (This is probably the right direction.)
- Where do they disagree? (This is the hard part of your decision.)
- What is the one thing you haven't said out loud yet?

### 5. Recommendation

State a clear recommendation. Format:

> **Do X.** The strongest reason is Y. The strongest objection is Z. This recommendation holds unless W is true — in which case reconsider.

### 6. Blind spot

Name what this entire analysis may have missed. One sentence.

## Output shape

- 400-800 words total
- Scannable with headers
- Each lens gets 3-5 sentences, not a paragraph
- End with a clear recommendation, not "it depends"
